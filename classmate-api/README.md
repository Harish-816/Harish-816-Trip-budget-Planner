# Classmate API: Fare Splitting Service

This service acts as the "Classmate's Code". It focuses purely on the algorithm.
**Responsibility**: Receive a list of paid expenses and return who owes whom.

## 1. Prerequisites
- Node.js installed

## 2. Code Structure
- `index.js`: Main server file
- `splitLogic.js`: The algorithm
- `package.json`: Dependencies

## 3. How to Run
`node index.js`
Runs on Port: **4000** (Default)
